## Features
- JWT Authentication
- PDF + Image upload with drag-and-drop
- Text extraction (pdf-parse + Tesseract OCR)
- AI itinerary generation (Google Gemini 2.5 Flash)
- Itinerary history + delete
- Public sharing with unique link
- Fully responsive UI
